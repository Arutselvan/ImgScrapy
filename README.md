# pyImgScraper
A simple image scraper written in python

### Dependencies

pyImgScraper uses lxml library. To install

```
pip install lxml
```

### Using the script

For running the script, clone the repo and make it as the current directory,then

```
python pyImgScraper.py http://example.com
```
